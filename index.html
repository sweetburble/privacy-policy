시행일: 2025년 6월 12일

1. 개인정보처리방침 개요

본 개인정보처리방침은 n8n 워크플로우 자동화 도구를 셀프호스팅 환경에서 운영하면서 수집되는 개인정보의 처리에 관한 사항을 규정합니다. 본 서비스는 개인이 운영하는 자동화 워크플로우 시스템으로, 사용자의 개인정보 보호를 최우선으로 합니다.

2. 개인정보 수집 항목 및 수집 방법

수집하는 개인정보 항목

필수 항목:

  • Gmail 계정 정보 (이메일 주소, 계정 ID)
  • Google Drive 파일 접근 권한
  • 워크플로우 실행을 위한 최소한의 계정 정보

자동 수집 항목:

  • 접속 IP 주소
  • 워크플로우 실행 상태 정보
  • 오류 코드 및 메시지 (페이로드 데이터 제외)

수집하지 않는 정보

본 서비스는 다음 정보를 수집하지 않습니다:

  • 개인 식별 정보 (IP 주소 제외)
  • 자격 증명 정보 (비밀번호, API 키 등)
  • 실행 데이터의 실제 내용
  • 민감한 설정 정보

3. 개인정보 수집 및 이용 목적

주요 목적:

  • n8n 워크플로우 자동화 서비스 제공
  • Gmail 및 Google Drive와의 연동을 통한 자동화 작업 수행
  • 서비스 오류 진단 및 개선
  • 워크플로우 실행 상태 모니터링

세부 이용 목적:

  • 이메일 자동 처리 및 분류
  • 파일 자동 백업 및 동기화
  • 데이터 변환 및 전송 자동화
  • 시스템 안정성 및 성능 향상

4. 개인정보 처리 및 보유 기간

보유 기간

  • 워크플로우 실행 데이터: 사용자 설정에 따라 자동 삭제 (기본 7일)
  • OAuth 인증 정보: 서비스 이용 기간 동안
  • 오류 로그: 최대 30일 후 자동 삭제

자동 삭제 설정

본 시스템은 EXECUTIONS_DATA_MAX_AGE 환경 변수를 통해 실행 데이터를 자동으로 삭제하도록 설정되어 있습니다.

5. 개인정보 처리 위탁

본 서비스는 셀프호스팅 환경에서 운영되므로 개인정보 처리를 외부에 위탁하지 않습니다. 모든 데이터는 자체 인프라 내에서 처리됩니다.

외부 연동 서비스:

  • Google Gmail API (구글 계정 연동 목적)
  • Google Drive API (파일 접근 목적)

6. 개인정보 보안 조치

기술적 보안 조치

  • 네트워크 격리 및 접근 제어
  • 데이터 암호화 (전송 및 저장)
  • 정기적인 보안 취약점 점검
  • 시스템 접근 로그 모니터링

관리적 보안 조치

  • 개인정보 접근 권한 최소화
  • 정기적인 보안 정책 검토
  • 개인정보 처리 현황 기록 관리

7. 개인정보 처리 현황 통지

데이터 주권

  • 모든 워크플로우 데이터와 자격 증명은 자체 인프라 내에서만 처리됩니다
  • 명시적으로 설정하지 않는 한 외부로 데이터가 전송되지 않습니다
  • 워크플로우 설계 및 비즈니스 로직은 완전히 비공개로 유지됩니다

텔레메트리 데이터 수집 중단

본 시스템은 다음 환경 변수를 통해 텔레메트리 수집을 비활성화하였습니다:

  • N8N_DIAGNOSTICS_ENABLED=false
  • N8N_VERSION_NOTIFICATIONS_ENABLED=false
  • N8N_TEMPLATES_ENABLED=false

8. 정보주체의 권리

개인정보 열람권

  • 처리 중인 개인정보의 열람을 요청할 수 있습니다

개인정보 정정·삭제권

  • 개인정보의 정정이나 삭제를 요청할 수 있습니다
  • 워크플로우 실행 데이터는 사용자가 직접 삭제할 수 있습니다

개인정보 처리정지권

  • 개인정보 처리 중단을 요청할 수 있습니다
  • OAuth 연결 해제를 통해 데이터 수집을 중단할 수 있습니다
